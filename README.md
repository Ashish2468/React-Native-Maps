## Configuration Steps
1. Cloning the repository:

```
$ git clone https://github.com/Ashish2468/React-Native-Maps.git
```

2. Open the project:

`$ cd T-UI-Clone`

3. Now run the app on your connected device (using terminal):

`$ npm install` then
`$ npx react-native run-android`
